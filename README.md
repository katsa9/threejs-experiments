# threejs-experiments

## Build
* This project is built with WebPack. To bundle the files run:

```
npx webpack
```

## Running it
* This project is using the Servez local webserver. (https://greggman.github.io/servez/)
* To start the server run:

```
servez /path/to/entrypoint.html
```
